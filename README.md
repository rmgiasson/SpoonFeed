# SpoonFeed

to run backend: <br />
cd server <br />
npm install multer <br />
npm install express <br />
node server.js <br />

to run frontend: <br />
cd client <br />
cd src <br />
npm start <br />

to run mongodb: <br />
mongosh <br />
use spoonfeed; <br />
show collections; <br />

alternative if it has proper installs: <br />
choco install make (as administrator) <br />
make <br />
